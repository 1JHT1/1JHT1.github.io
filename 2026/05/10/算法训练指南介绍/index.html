后端笔试算法学习打卡清单

🔴 第一梯队（高频，必会）

基础数据结构

  • [ ] 数组：遍历、原地修改、前缀和
  • [x] 链表：反转、找环、找中点、合并有序链表
  • [ ] 哈希表：两数之和、统计频率、去重
  • [ ] 栈：括号匹配、单调栈
  • [ ] 队列：滑动窗口、BFS

字符串

  • [ ] 子串匹配（KMP 理解思想）
  • [ ] 括号匹配与生成
  • [ ] 字符串转整数（atoi）
  • [ ] 最长公共前缀/子串

🟠 第二梯队（中频，拉开差距）

双指针 / 滑动窗口

  • [ ] 左右指针（两数之和、反转数组）
  • [x] 快慢指针（环形链表）
  • [ ] 固定长度滑动窗口
  • [ ] 可变长度滑动窗口（最长无重复子串）

二分查找

  • [ ] 有序数组中查找边界
  • [ ] 旋转排序数组中的查找
  • [ ] 二分答案（最小值最大化、最大值最小化）

回溯算法

  • [ ] 全排列、子集、组合
  • [ ] 剪枝优化
  • [ ] N皇后、数独

贪心算法

  • [ ] 区间调度（无重叠区间）
  • [ ] 跳跃游戏
  • [ ] 分配饼干

基础动态规划

  • [ ] 爬楼梯（一维DP）
  • [ ] 打家劫舍（状态转移）
  • [ ] 背包问题（0-1、完全）
  • [ ] 最长公共子序列（LCS）
  • [ ] 最长递增子序列（LIS）

🟡 第三梯队（低频，提升上限）

二叉树

  • [ ] 前/中/后序遍历（递归+迭代）
  • [ ] 层序遍历（BFS）
  • [ ] 二叉搜索树（验证、查找、插入）
  • [ ] 最近公共祖先（LCA）

堆 / 优先队列

  • [ ] TopK 问题
  • [x] 合并 K 个有序链表
  • [ ] 数据流中的中位数

图

  • [ ] DFS / BFS 遍历
  • [ ] 岛屿数量（网格DFS）
  • [ ] 最短路径（Dijkstra）

并查集

  • [ ] 连通性问题
  • [ ] 朋友圈数量

前缀树（Trie）

  • [ ] 插入、搜索单词
  • [ ] 单词自动补全

🛠️ 工程细节（隐形高分项）

  • [ ] 熟练掌握 ACM 模式输入输出（Scanner / BufferedReader）
  • [ ] 代码规范：变量命名、缩进、无重复代码
  • [ ] 边界处理：空值、单元素、最大值、最小值
  • [ ] 时间复杂度与空间复杂度分析

📝 模拟与冲刺

  • [ ] 完成 LeetCode Hot 100 中至少 70 题
  • [ ] 在牛客网完成 5 套大厂笔试真题（限时）
  • [ ] 整理自己的错题本和常见模板（二分、回溯、并查集等）
  • [ ] SQL 复习：连表查询、聚合、窗口函数（针对有 SQL 题的公司）
