1.前缀和技术

前缀和：前缀和是一种预处理技术，通过预先计算数组从起点到每个位置的累加和（一般不包含自己），从而能够快速（O(1) 时间）查询任意子数组的和。用空间换查询时间

一维数组前缀和的构造和查询：

前缀和是算法竞赛和面试中最基础也最实用的技巧之一，一维用于区间和，二维用于子矩阵和，结合哈希表可高效解决子数组统计问题。

典型应用场景
问题类型 描述
子数组和查询 给定数组，多次询问不同子区间的和。
统计和为 k 的子数组个数 利用前缀和 + 哈希表，O(n) 解决。
连续子数组的最大/最小和 结合 Kadane 算法或前缀和极值。
二维区域和检索 LeetCode 304. Range Sum Query 2D – Immutable。
差分数组的逆操作 对原数组多次区间加减后，通过前缀和恢复原值。

3.滚动数组

滚动数组是一种空间优化技巧，通过重复使用固定大小的数组（通常只保留前后几步的状态），将动态规划的空间复杂度从 O(n²) 或 O(n) 降为 O(1) 或 O(k)。例如斐波那契数列只需两个变量交替更新，背包问题用一维数组逆序滚动代替二维数组。
