1.Java的类里有什么？

类的主要成分有成员变量、构造方法（带参、无参）、成员方法、getter、setter方法（封装）

组成部分 作用 示例
成员变量 存储对象的状态（数据） private String brand;
构造方法 创建对象时初始化成员变量 public Car(String b) { brand = b; }
方法 定义对象的行为或功能 public void start() { ... }
内部类 定义在另一个类内部的类 class Engine { }
静态成员 属于类本身，而不是对象 public static int wheelCount = 4;
